⸻

📌 על הפרויקט

🔹 האתר מאפשר ניהול ושמירה של מתכונים אישיים ישירות על המכשיר, ללא צורך בחיבור לשרת.
🔹 ניתן להוסיף, לערוך, למחוק, ולשתף מתכונים בפורמט טקסטואלי או JSON.
🔹 תומך במצב כהה/בהיר, חיפוש מתקדם, קטגוריות מותאמות אישית ועוד.
🔹 הקוד פתוח לשימוש ולשינוי, אך יש לתת קרדיט ליוצר המקורי.

⸻

📂 תכונות עיקריות

📖 ניהול מתכונים

✔️ הוספת מתכון חדש – ניתן להזין שם, קטגוריה, רשימת מרכיבים והוראות הכנה.
✔️ עריכת מתכון קיים – ניתן לערוך כל פרט במתכון ולשמור אותו מחדש.
✔️ מחיקת מתכון – אפשר למחוק מתכון בודד או את כל המאגר בלחיצה אחת.
✔️ שמירת מתכונים מקומית – כל הנתונים נשמרים במכשיר בלבד, ללא שימוש בבסיס נתונים חיצוני.

🔍 חיפוש וסינון

✔️ חיפוש חכם – ניתן לחפש מתכונים לפי שם או מרכיב ספציפי.
✔️ סינון לפי קטגוריות – ניתן להציג רק מתכונים מקטגוריה מסוימת.
✔️ ניהול קטגוריות – ניתן להוסיף, לערוך ולמחוק קטגוריות בהתאמה אישית.

📤 ייבוא וייצוא מתכונים

✔️ שיתוף מתכון כטקסט או JSON – ניתן להעתיק מתכון או לשמור אותו כקובץ JSON.
✔️ ייבוא מתכון בודד – ניתן להעלות קובץ JSON ולשמור את המתכון במערכת.
✔️ ייצוא כל המתכונים – ניתן לגבות את כל המתכונים בקובץ JSON ולהעלות אותם במכשיר אחר.

⭐ מועדפים

✔️ סימון מתכון כמועדף – ניתן לסמן מתכונים נבחרים לגישה מהירה.
✔️ סינון מועדפים – אפשר להציג רק מתכונים שסומנו כמועדפים.

🌙 מצב כהה ובהיר

✔️ מעבר בין מצב כהה/בהיר – מאפשר התאמה אישית לפי העדפה, עם שמירת ההגדרה בלוקאלי.
